TARGO Dataset

We present a novel dataset for 6DoF grasp pose estimation, simulated using PyBullet. To the best of our knowledge, this is the first target-driven dataset that includes an analysis of occlusion levels. Our dataset comprises two types of scenes: cluttered scenes (denoted as 'c') and single-object scenes (denoted as 's').

The TARGO_Synthetic training dataset includes grasp label files, scene files, and a mesh pose dictionary. The grasp files contain the scene ID, corresponding grasp poses, and occlusion levels. The mesh pose dictionary includes mesh files and their corresponding poses, which can be used to reconstruct and simulate the scene in the PyBullet environment. The scene files provide inputs for our model and baseline models such as GIGA, VGN, and GIGA-HR. Specifically, these files contain monocular depth images, target object masks, TSDF grids of the scene, and depth backprojected target points, among other features.
